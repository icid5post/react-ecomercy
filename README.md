# Simple react-ecomercy shop

## Build Setup:

``` bash
# Download repository:
git clone https://github.com/icid5post/react-ecomercy.git react-ecomercy

# Go to the app:
cd react-ecomercy

# Install dependencies:
yarn install

# Server with hot reload at http://localhost:3000/
npm run dev

# Output will be at dist/ folder
npm run build
```
